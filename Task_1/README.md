# 📚 Web Scraping with BeautifulSoup

## 📌 Project Overview

This project demonstrates web scraping using Python and BeautifulSoup. The data is collected from the **Books to Scrape** website, a public website designed for practicing web scraping.

The scraper automatically extracts book information from all available pages, creates a structured dataset, and exports the data into a CSV file for further analysis.

## 🎯 Objective

- Learn web scraping using Python.
- Understand HTML structure and webpage navigation.
- Extract data from public websites.
- Create a custom dataset for analysis.
- Export the collected data into CSV format.


## 🛠️ Technologies Used

- Python
- Requests
- BeautifulSoup (bs4)
- Pandas
- Jupyter Notebook

## 📂 Dataset Information

The dataset contains the following fields:

- Book Title
- Price
- Rating
- Availability
- Product URL

Total Records:
- **1000 Books**

Output File:
- **books.csv**

## 🔄 Project Workflow

1. Import required Python libraries.
2. Connect to the website using Requests.
3. Parse HTML using BeautifulSoup.
4. Locate book information using HTML elements.
5. Extract:
   - Title
   - Price
   - Rating
   - Availability
   - Product URL
6. Navigate through all website pages.
7. Store data in a Pandas DataFrame.
8. Export the dataset as CSV.
